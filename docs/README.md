# Documentation Index

This folder contains complete project documentation split into dedicated files:

1. `PROJECT_STRUCTURE.md` — complete repository structure and module breakdown.
2. `PROJECT_DETAILS.md` — product behavior, features, and current scope.
3. `TECH_STACK.md` — frontend/backend dependencies, protocols, and tooling.
4. `ARCHITECTURE.md` — runtime flow, component design, and data architecture.
5. `PROJECT_INFORMATION.md` — goals, stakeholders, use cases, roadmap, metrics.
6. `HACKATHON_PPT_GUIDE.md` — slide-by-slide pitch deck content and demo script.

Use this index as the single starting point for onboarding judges, teammates, and reviewers.
